## Template Variables (All Supported)

### User Input Variables
- `{{ party_b_legal_name }}` - Company name
- `{{ party_b_registered_address }}` - Registered address
- `{{ party_b_delivery_address }}` - Delivery address
- `{{ party_b_business_form }}` - Business form (CV/PT/Others)
- `{{ customer_partner_code }}` - CVCODE number
- `{{ party_b_authorized_representative_name }}` - Representative name
- `{{ party_b_authorized_representative_title }}` - Representative title

### Auto-Calculated Variables
- `{{ contract_effective_start_date }}` - Contract start date
- `{{ contract_effective_end_date }}` - Contract end date
- `{{ total_annual_purchase_target }}` - Annual purchase amount
- `{{ q1_period }}, {{ q2_period }}, {{ q3_period }}, {{ q4_period }}` - Quarterly periods
- `{{ q1_sales_target }}, {{ q2_sales_target }}, {{ q3_sales_target }}, {{ q4_sales_target }}` - Quarterly targets
